# battle_ship
## [レーダー作戦ゲーム](https://ja.wikipedia.org/wiki/%E6%B5%B7%E6%88%A6%E3%82%B2%E3%83%BC%E3%83%A0)
ルールは以下の通り
- プレイヤーは二人
- マス目の設定は縦横10マス。
- 軍艦に耐久力は設定されない。その代わりに大きさが設定され、空母（5マス）、戦艦（4マス）、巡洋艦（3マス）、潜水艦（3マス）、駆逐艦（2マス）となっている。軍艦のマス目全部に攻撃を受けると沈没となる。
- 軍艦は移動できない。
